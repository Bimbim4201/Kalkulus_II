Bentuk Fungsi

Fungsi Eksplisit

Fungsi Eksplisit adalah fungsi dimana antara variabel bebas dan tak bebas (terikat) dapat dibedakan dengan jelas.
Contoh: y = 2x + 1

Fungsi Implisi

Fungsi Implisit adalah fungsi dimana antara variabel bebas dan variabel tak bebas (terikat) tidak dapat dibedakan dengan jelas.
Contoh: 2y + 3x –1 = 0

Sebelum masuk ke pembahasan fungsi linear, teman-teman harus tau dulu nih apa itu fungsi?
Fungsi adalah Suatu bentuk hubungan matematis yang menyatakan hubungan setiap anggota domain dengan satu dan hanya satu anggota codomainnya

FUNGSI LINEAR diartikan sebagai suatu fungsi yang membentuk grafik garis lurus.
Fungsi linear merupakan fungsi yang mendapat pangkat tertinggi variabelnya sama dengan satu.

Bentuk Persamaan Umumnya:

f(x) = ax + bx + c
Contoh: 3x + 2y – 5 = 0
Berdasarkan bentuk umum persamaan Fungsi Linier:
AX + BY + C = 0
Dapat diubah menjadi:
Y = mX + k


Dimana:
m = -A/B adalah gradien
k = -C/B adalah konstanta
X = variabel bebas
Y = variabel terikat
Menggambar Grafik Fungsi Linier

Contoh : y = 2x + 6
Langkah-langkah menggambarkannya:

• Titik potong dengan sumbu X
y = 0
0 = 2x + 6
x = -3
• Titik potong dengan sumbu Y
x = 0
y = 2.0 +6
y = 6
Hubungkan kedua titik potong tersebut.
Menentukan gradien

• Apabila fungsi linier melalui dua titik
A(x1, y1) dan B(x2, y2)
maka gradiennya:
m = (y2 - y1) / (x2 - x1)

• Apabila Grafik fungsi linier mempunyai persamaan
AX + BY + C = 0
maka gradiennya
m = -A/B


Kemiringan grafik fungsi linier :
• Apabila gradien m > 0 maka grafiknya miring ke kanan
• Apabila gradien m < 0 maka grafiknya miring ke kiri
• Apabila gradien m = 0 maka grafiknya sejajar sumbu X
• Apabila gradien m = ~ maka grafiknya sejajar sumbu Y

Contoh Soal Fungsi Linear

Contoh 1

Terdapat suatu fungsi linear adalah f(x) = 10x + b. Tentukanlah bentuk fungsi tersebut jika diketahui f(5) = 15.
Penyelesaian:
f(x) = 10x + b
f(5) = 10 x 5 + b = 15
15 = 10 x 5 + b
b = 15 – 50
b = -35
f(x) = 10x – 35

Contoh 2

Silahan tentukan wujud dari persamaan garis lurus ini apabila didapatkan f (4) =8. Dan diketahui f (x) = 6x + b
Pembahasan:
f (x) = 6x + b
F(4) = (6.4) + b = 8
B = 8 – 24
B = -16
Sehingga didapatkan bentuk dari fungsi linear : f (x) = 4x-16.

Contoh 3

F(x) =10x + b. Bagaimana bentuk dari persamaan garis lurusnya apabila f (5) = 15?
Pembahasan :
f (x) = 10x + b
f (5) = (10×5) + b
f(5) = (10×5) + b = 15
15 = (10×5) + b
Didapatkan b = 15 – 50 =-35 sehingga b =-35.
Jika dibuatkan bentuk persamaan garis lurusnya menjadi f(x) 10x-35
