Materi Fungsi Eksponen

1. Pengertian Fungsi Eksponen

Fungsi eksponen adalah fungsi matematika di mana variabelnya menjadi pangkat (eksponen) dari suatu bilangan tetap (basis). Bentuk umumnya adalah: f(x) = a^x

  • a adalah basis (bilangan pokok), dengan syarat a > 0 dan a ≠ 1.
  • x adalah eksponen (variabel).

Jika a > 1, fungsi bersifat menaik (increasing). Jika 0 < a < 1, fungsi bersifat menurun (decreasing).

2. Sifat-sifat Fungsi Eksponen

  • Sifat 1: a^0 = 1
  • Sifat 2: a^x * a^y = a^(x+y)
  • Sifat 3: a^x / a^y = a^(x-y)
  • Sifat 4: (a^x)^y = a^(x*y)
  • Sifat 5: a^(-x) = 1 / a^x

3. Grafik Fungsi Eksponen

Grafik fungsi eksponen memiliki ciri khas yang dapat dibedakan berdasarkan nilai basisnya:

  • Jika a > 1: Grafik akan naik dari kiri ke kanan, semakin tinggi seiring bertambahnya nilai x.
  • Jika 0 < a < 1: Grafik akan turun dari kiri ke kanan, semakin mendekati 0 seiring bertambahnya nilai x.

4. Aplikasi Fungsi Eksponen

Fungsi eksponen punya banyak aplikasi dalam kehidupan sehari-hari. Berikut beberapa contoh penerapannya:

  • Pertumbuhan Populasi: Menggunakan rumus P(t) = P0 * e^(rt).
  • Pertumbuhan Bunga Bank: Menggunakan rumus A = P * (1 + r/n)^(nt).
  • Peluruhan Radioaktif: Menggunakan rumus N(t) = N0 * e^(-kt).

5. Contoh Soal dan Pembahasan

Contoh 1:

Diketahui fungsi f(x) = 3^x. Tentukan nilai dari f(2) dan f(-1).
Pembahasan:
f(2) = 3^2 = 9
f(-1) = 3^(-1) = 1/3

Contoh 2:

Jumlah penduduk suatu desa pada tahun 2020 adalah 1.000 orang. Jika tingkat pertumbuhan penduduk adalah 5% per tahun, berapakah perkiraan jumlah penduduk pada tahun 2025?
Pembahasan:
Gunakan rumus pertumbuhan eksponensial: P(t) = P0 * (1 + r)^t
P(5) = 1000 * (1 + 0.05)^5 ≈ 1276
Jadi, pada tahun 2025, jumlah penduduk diperkirakan sekitar 1.276 orang.
